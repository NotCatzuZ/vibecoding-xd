# Vibecoding Battle

Et actionfylt spill hvor du styrer en bærbar PC og unngår fiender mens du samler oppdateringer!

## Hvordan spille

### Kontroller
- **WASD** eller **piltaster**: Beveg spilleren
- **Q**: Aktiver ultimate angrep når den er fulladet (rydder fiender rundt deg)

### Mål
- Unngå blå bølger, lilla virus og gule hamre som kommer fra alle sider
- Samle grønne **oppdateringer** for å fylle ultimate måleren og få liv
- Ultimate lades ved å unngå fiender og samle oppdateringer
- Spillet blir vanskeligere jo lenger du overlever!

### Spillmekanikk
- **Liv**: Starter med 3 liv, mister liv ved å treffe fiender
- **Livstap**: Livet synker raskt hvis du ikke samler oppdateringer på 8+ sekunder
- **Poeng**: Øker automatisk over tid
- **Nivå**: Øker hvert 8. sekund, gjør fiender raskere og flere
- **Highscore**: Lagres lokalt med ditt navn når du dør

### Fiender
- **Blå bølger**: Sakte, kommer fra alle sider
- **Lilla virus**: Store og raske, kommer fra alle sider
- **Gule hamre**: Raske og uforutsigbare, kommer fra alle sider

## Hvordan kjøre

Åpne `index.html` i en nettleser som støtter moderne JavaScript og Canvas API.

## GitHub Pages

Spillet er deployet via GitHub Pages og kan spilles direkte i nettleseren på: [https://notcatzuz.github.io/vibecoding-xd/](https://notcatzuz.github.io/vibecoding-xd/)

## Utvikling

Bygget med ren HTML5, CSS3 og JavaScript ES6+.
- Canvas API for rendering
- Web Audio API for lydeffekter
- localStorage for highscores
